Deze code is zo opgebouwd dat de auto rijdt wanneer er een knopje (aangesloten op A0) word ingedrukt.
Als er ook een ultrasone sensor (trig A4, echo A5) word aangesloten reageert de auto op obstakels.
Als er een piezo speaker wordt aangesloten op 9 klinkt er een toon wanneer de auto word gestart.
Als er twee leds worden aangesloten op (A1 en A2) dan weergeven deze wanneer er een object word waargenomen.

This code is constructed in such a way that the car drives when a button (connected to A0) is pressed.
If an ultrasonic sensor (trig A4, echo A5) is also connected, the car reacts to obstacles.
If a piezo speaker is connected to 9 a tone will sound when the car is started.
If two LEDs are connected to (A1 and A2), they will show when an object is detected.

the robotcar kit and AFMotor driver are being used as the base of this robot.
